# 🔗 ChainBall Physics Simulation

## 📌 Overview
This project is a Blender physics simulation featuring a chain of connected objects ending with a heavy ball. The system demonstrates realistic rigid body physics, including collision, gravity, and momentum transfer.

When the ball moves, it creates a chain reaction effect, showing how forces travel through connected objects.

## 🎮 How to Run
1. Open the `.blend` file in Blender
2. Go to **Timeline**
3. Press **Play (Spacebar)** to run the simulation
4. Adjust physics settings if needed in the **Rigid Body panel**

---


## 🚀 Notes
- Make sure rigid body world is enabled
- Apply transforms before simulation (`Ctrl + A`)
- Bake simulation for stable results
